## Sección 2 — El problema

### Eyebrow
```
El problema real
```

### Titular
```
Los errores de aplicación
cuestan caro y se detectan tarde.
```

### Cuerpo
```
Cuando terminás un trabajo, el archivo queda en la pantalla. Nadie lo analiza. 
Nadie sabe si se solapó, si la dosis varió, si hubo zonas sin cubrir. 
Hasta que el productor se queja o se cuentan los kilos sobrantes.
```

### Items de problema

**Item 1**
```
Título:   Solapamientos no detectados
Cuerpo:   Pasadas que se pisan implican doble insumo gastado y sin rendimiento extra.
```

**Item 2**
```
Título:   Sin evidencia del trabajo realizado
Cuerpo:   El contratista no puede demostrarle al productor que cumplió con la prescripción.
```

**Item 3**
```
Título:   Desvíos de dosis invisibles
Cuerpo:   Zonas con subdosis o sobredosis que afectan el resultado del cultivo sin que nadie lo sepa.
```

### Quote / caso real

## Sección 3 — La plataforma

### Eyebrow
```
La plataforma
```

### Titular
```
Todo lo que necesitás
para analizar un trabajo.
```

### Subtítulo
```
Subís el archivo de la pantalla, la plataforma lo procesa y en minutos tenés 
un análisis completo. Sin instalaciones, sin configuración, desde cualquier dispositivo.
```

### Features

**Feature 1**
```
Título:       Mapa del trabajo
Descripción:  Visualizá cada pasada sobre imagen satelital. Coloreado automático 
              por velocidad, dosis aplicada o desvío de prescripción.
Plan:         Incluido (free)
```

**Feature 2**
```
Título:       Detección de solapamientos
Descripción:  Calcula automáticamente las zonas de doble aplicación: área en hectáreas 
              y kilos estimados de producto gastado de más.
Plan:         Incluido (free)
```

**Feature 3**
```
Título:       Prescripción vs Realidad
Descripción:  Compará lo que indicó el agrónomo con lo que ejecutó la máquina. 
              Zona por zona, con porcentaje de cumplimiento.
Plan:         Incluido (free)
```

**Feature 4**
```
Título:       Estadísticas completas
Descripción:  Velocidad mínima, promedio y máxima. Dosis real vs recomendada. 
              Histogramas de distribución. Todo en una pantalla.
Plan:         Incluido (free)
```

**Feature 5**
```
Título:       Informe PDF técnico
Descripción:  Generá un reporte listo para entregar al productor: mapas, estadísticas 
              y resumen ejecutivo. Con logo de tu empresa.
Plan:         Plan Pro
```

**Feature 6**
```
Título:       Historial de campaña
Descripción:  Guardá todos los trabajos de la temporada y comparalos. 
              Detectá patrones, equipos con desvíos y zonas problemáticas.
Plan:         Plan Pro
```

## Sección 6 — Cómo funciona

### Eyebrow
```
El proceso
```

### Titular
```
Cuatro pasos, cinco minutos.
```

### Subtítulo
```
No hace falta saber de SIG ni tener un software especial. 
Si podés subir un archivo, podés analizar un trabajo.
```

### Pasos

**Paso 1**
```
Título:   Subís el archivo
Cuerpo:   Exportás el trabajo desde la pantalla de la máquina 
          (KML, Shapefile o GeoJSON) y lo subís a la plataforma.
```

**Paso 2**
```
Título:   Se procesa automáticamente
Cuerpo:   La plataforma detecta los campos (dosis, velocidad, prescripción) 
          y calcula todos los análisis en segundo plano.
```

**Paso 3**
```
Título:   Analizás los resultados
Cuerpo:   Ves el mapa, las estadísticas, los solapamientos y el cumplimiento 
          de la prescripción en una sola pantalla.
```

**Paso 4**
```
Título:   Generás el informe
Cuerpo:   Exportás el PDF técnico y se lo mandás al productor o lo guardás 
          en tu historial de campaña.
```
